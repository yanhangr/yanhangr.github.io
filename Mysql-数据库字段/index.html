数据库字段规范

主键ID （unsigned int 自增）

可存储0 ~ 4294967295之间，占4个字节（byte），32位；单表最大可存储42亿数据完全够用
相比 unsigned bigint完全用不上，占用空间又大，其实单表超过千万条数据，就应该分表了
